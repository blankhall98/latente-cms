How To Use The Dashboard

  1. 1 Select the correct project. Use Projects when you manage more than one site. The active project appears next to the Blank CMS logo.
  2. 2 Open Pages and choose the content to edit. Pages are grouped by section. Open the page, update the fields, and keep media inside the provided controls.
  3. 3 Save draft changes first. Saving keeps the work inside the dashboard without making it public immediately.
  4. 4 Review before publishing. Check copy, links, images, and repeated items. Publishing updates the JSON consumed by the public website.
  5. 5 Ask for help when something looks risky. Use the support form for publish errors, missing media, login issues, or content that does not appear after publishing.
Avoid changing technical fields such as slugs, schema-specific keys, or shared navigation content unless you are sure the website expects that change.
